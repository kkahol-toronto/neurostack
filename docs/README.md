- **[SETUP_GUIDE.md](SETUP_GUIDE.md)** - Comprehensive setup guide for Azure resources and local services

## 🚀 **Getting Started**

1. **Read the Setup Guide** - Start with [SETUP_GUIDE.md](SETUP_GUIDE.md) for complete setup instructions
2. **Use the Setup Script** - Run `../scripts/setup_neurostack.sh` for automated setup
3. **Verify Your Setup** - Run `python ../debug/check_neurostack_setup.py` to verify configuration
